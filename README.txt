A plugin for testing the performance of the themes, plugins and scripts of your Wordpress site

== Description ==

This plugin help you to test and check the performance of the plugins, themes and scripts you have
installed in your wordpress site.

Its bring easy to read tables and charts that show what theme, plugin or page are the ones that are slowing down your website.

Its comes in several languages that guaranteed you will know how to use it or what is the problem in you site.

It makes it very simple to locate any bottleneck problem in your themes or plugins in order to solve it and speed up your website.

Save your scans and profiles in a history table to later use in your performance reports.

Install and active the plugin so you can start checking your wordpress site right away.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `profiling-tool-for-wp.zip` to the `/wp-content/plugins/` directory or directly add it from the wordpress plugins section.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the main plugin page throught the new added link in the main wordpress sidebar


= Features =

* Plugins & themes performance profiling.
* Page performance profiling.
* Saved profiling history scans.
* Multi-language (EN, ES, GL).
* Testing single themes or plugins.
* Testing multiple pages charging times at once.
